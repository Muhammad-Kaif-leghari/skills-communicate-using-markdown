## Introduction
This is a simple markdown file where we demonstrate how to use headers.

### Features

- Lightweight syntax

Markdown can be used for:
1. Documentation

##### Summary
Headers in markdown are created with `#` characters followed by a space. Use from one `#` (largest) to six `######` (smallest).
